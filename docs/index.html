    Memory Conversion

    Quick and easy conversion between memory values.

    Installation

    1. Add the dependency to your shard.yml:

    `yaml dependencies:

     memory_conversion:
   github: dscottboggs/memory_conversion

    `

    1. Run shards install

    Usage

    require "memory_conversion"

1.terabyte.to_kilobits
